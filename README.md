## ⚙️ Installation
### Prerequisites
- npm or yarn

### Install Dependencies
```bash
npm install
```

## 🚀 Running
```bash
npm run dev
```

Then open your browser at: http://localhost:5173

- For testing the web, you can open <b>mockUsers.json</b> and choose one of accounts to sign in.
